# 项目结构说明

## 目录结构
```
ez-admin-springboot4/
├── src/main/java/com/ezadimn/          # 主要Java源代码
│   ├── EzAdminSpringboot4Application.java  # 启动类
│   ├── config/                           # 配置类
│   ├── controller/                      # 控制器层
│   │   └── UserController.java          # 用户管理控制器
│   ├── dto/                             # 数据传输对象
│   │   └── UserDTO.java                 # 用户DTO
│   ├── entity/                          # 实体类
│   │   └── User.java                    # 用户实体
│   ├── repository/                      # 数据访问层
│   │   └── UserRepository.java          # 用户数据访问接口
│   └── service/                         # 服务层
│       └── UserService.java             # 用户服务类
├── src/main/resources/                  # 资源文件
│   └── application.yaml                 # 应用配置文件
├── pom.xml                             # Maven配置文件
└── README.md                           # 项目说明文档
```

## 架构分层

### 1. 控制器层 (Controller)
- **职责**：处理HTTP请求，返回响应数据
- **特点**：RESTful风格，统一响应格式，异常处理

### 2. 服务层 (Service)
- **职责**：业务逻辑处理，事务管理
- **特点**：包含业务规则，数据校验，异常抛出

### 3. 数据访问层 (Repository)
- **职责**：数据库操作，数据持久化
- **特点**：继承JpaRepository，提供CRUD方法

### 4. 实体层 (Entity)
- **职责**：数据模型定义，数据库表映射
- **特点**：JPA注解，Lombok简化代码

## 核心功能
